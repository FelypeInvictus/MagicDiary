# Run

- `sudo chmod +x MagicDiary.sh`
- `bash MagicDiary.sh`

# Dependencies

bash | bat | neovim

## Debian

- `apt install bat nvim bash -y`

## ArchLinux

- `sudo pacman -S bat nvim bash --noconfirm`

Other OS: Install the dependencies in the respective repositories.
